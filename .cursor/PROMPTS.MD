```markdown

**Дата/время:** YYYY-MM-DD HH:mm

**Запрос:**
<полный текст>

**Результат / решения:**
- ...
```

**Идеи для следующих итераций (не обязательно сразу):**

1. История действий по заявке (audit log).
2. Улучшение UI и сообщений об ошибках.
3. Скрипт `scripts/race_test.sh` / `race_test.ps1` для демонстрации 409.
4. Деплой на Vercel + внешняя БД (Postgres), если SQLite на Vercel не подходит.
5. E2E-тесты (Playwright).
