# Участие в проекте

Репозиторий: **advayta108/ts-response-service-example**

## Как помочь

1. Форкните репозиторий и создайте ветку от `main`.
2. Локально: `bun install`, `bun run db:migrate`, `bun run db:seed`, `bun run dev`.
3. Перед PR: `bun test`, `bun run build`.
4. Опишите изменения в PR; для багов приложите шаги воспроизведения.
